    \section{Symbolic Solution for Limit Continuous
Poker}\label{symbolic-solution-for-limit-continuous-poker}

This notebook derives the Nash equilibrium of LCP using transformed
coordinates: - \(r = L/(1+L)\) (minimum pot odds) - \(t = 1/(1+U)\) (pot
fraction at max bet)

The solution provides closed-form expressions for all strategic
components.

\begin{enumerate}
